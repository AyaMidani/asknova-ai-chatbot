# AskNova AI Chatbot

AskNova is a full-stack AI chatbot application that allows users to interact with an AI assistant through a responsive chat interface. The project includes reusable UI components, API integration, conversation handling, loading states, and production-ready configuration.

## Tech Stack

Frontend: React, JavaScript, HTML5, CSS3, TailwindCSS
Backend: Node.js, Express.js
Database: MongoDB
API: OpenAI API
Tools: Git, GitHub, Postman, Render/Vercel

## Features

Responsive AI chat interface
OpenAI API integration
Conversation history
Loading and error states
REST API backend
Environment-based API key management
Clean and reusable React components
Production-ready deployment setup

## Project Purpose

This project was created to practice building AI-powered full-stack applications using modern JavaScript technologies. It focuses on clean UI development, backend API integration, secure API key handling, and scalable project structure.
